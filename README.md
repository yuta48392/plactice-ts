# plactice-ts


このリポジトリでは、TypeScript と Express を使った API 開発の練習を行う。
小規模な CRUD 機能を中心に、実装・構成・Docker 化までを段階的にまとめる。

## 概要
- Express を使った API サーバー
- SQLite を利用したシンプルなデータ永続化
- Docker での起動環境構築
- 開発手順・学習ログを README にまとめる

## 目的
TypeScript を中心とした Web API 開発の手順を身につけるためのアウトプット。
フレームワークの理解、非同期処理、Docker を含む基本的な構成を学習する。

## 使用技術
- TypeScript
- Express
- SQLite
- Docker
